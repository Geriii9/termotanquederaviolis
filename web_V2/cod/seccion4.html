Esto es el CSS de la página principal 



        .row {
            display: flex;
            align-items: center;
            justify-content: center;
            margin: 2%;
          }
          
          .column {
            padding: 4% 2%;
            margin: 2%;
            background: white;
          
            width: 40%;
            border: 1px solid #d6d6d6;
            box-shadow: 0 2px 3px 0px rgba(0, 0, 0, 0.25);
            border-radius: 3px;
            transition: 0.2s all;
          }
          
          .row:hover .column {
            filter: blur(3px);
            opacity: 0.5;
            transform: scale(0.98);
            box-shadow: none;
          }
          
          .row:hover .column:hover {
            transform: scale(1);
            filter: blur(0px);
            opacity: 1;
            box-shadow: 0 8px 20px 0px rgba(0, 0, 0, 0.125);
          }

Este código es parte de la página principal. Este, divide las diferentes secciones en columnas, separandolas entre ellas. Al poner el cursor sobre una sección, esta se ampliará considerablemente. Al pulsar sobre una de las mencionadas secciones, la página te llevará a la sección seleccionada.
