Instalación Proxmox

Primero hemos buscado la ISO del sistema operativo proxmox, instalamos la versión proxmox-ve_7.2-1. Después instalamos el BalenaEtcher, seleccionamos la ISO y el lugar donde queremos instalar la ISO que era un pendrive y flasheamos la ISO. Cuando ya teníamos la ISO en el pendrive, conectamos el pendrive en el ordenador donde queremos instalar el sistema operativo. Después de conectar el pendrive abrimos la BIOS y seleccionamos el dispositivo de arranque, que en este caso es el USB que contiene la ISO. Una vez ya tenemos el ordenador encendido con el USB dentro configuramos paso por paso el nuevo SO(idioma, tipo de teclado, IPs...). Una vez con todo esto hecho ya podemos pasar al siguiente paso.



Configuración del servidor en Proxmox


Una vez Proxmox está instalado nos dará una IP, la cual será la que deberemos poner en el buscador de nuestro navegador para acceder a la pantalla de configuración, allí es dónde configuraremos nuestro servidor para la página web. Lo que nosotros hemos hecho a sido crear un contenedor, ya que es más fácil de manejar, i dentro poner una imagen del sistema operativo ubuntu. Ya dentro del contenedor instalamos el apache, el cuál es el siguiente paso.
