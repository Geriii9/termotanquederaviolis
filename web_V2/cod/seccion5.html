Proxmox

Proxmox es un sistema operativo de código abierto que puede contener máquinas virtuales y contenedores para utilizarlas fácilmente, es por esto que es uno de los mejores SO para crear páginas web como estamos haciendo.


BalenaEtcher

Es una herramienta importante para instalar sistemas operativos en ordenadores, ya que este programa te permite flasehar la ISO(la imagen del sistema operativo) en un USB para más tarde usarla en otro ordenador y instalarle el SO prevamiente instalado en el USB.


Apache

Este es el servicio qué hemos usado para crear la página web, ya que nos permite, a través de una IP dada por la máquina, esta página sea accesible a todo el mundo


GIT

Herramienta que hemos usado para subir nuestro trabajo a la nube. Usando una serie de comandos podemos realizar dicha acción y, a la vez, trabajar con nuestros compañeros sin molestarlos. Más tarde, esta herramienta la hemos usado para subir nuestro trabajo al servidor.
