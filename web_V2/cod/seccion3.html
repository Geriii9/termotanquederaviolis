Instalación de la píldora Apache

Para instalar esta píldora dentro de nuestro contenedor pusimos el comando “sudo apt install apache2”, no hay mucho más que decir en esta parte así que vamos a configurar esta píldora.

Configuración de la píldora Apache

Configurar esta píldora ha sido complicado para nosotros, principalmente porque daba muchos errores a la hora de poner el comando, además no guardaba los datos proporcionados al documento editado, pero al final conseguimos poder editar el apache con estos comandos: “cd /var/www/gci/” y “nano index.html.” Una vez dentro del archivo, ponemos el código html. Una vez terminado el código, guardamos y cerramos el documento y nos disponemos a editar el archivo de configuración, concretamente el serveradmin y el nombre.
