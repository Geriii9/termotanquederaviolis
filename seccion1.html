    hello man

    Constará de varias páginas explicando como hemos creado un servidor a partir del uso de Proxmox y Apache

    Instalación Proxmox

    Primero hemos buscado la ISO del sistema operativo proxmox, instalamos la version proxmox-ve_7.2-1. Despues instalamos el BalenaEtcher, seleccionamos la ISO y el lugar donde queriamos instalar la ISO que era un pendrive y flasheamos la ISO. Cuando ya teniamos la ISO en el pedrive, conectamos el pendrive en el ordenador donde queriamos instalar el sistema operativo. Después de conectar el pendrive abrimos la BIOS y seleccionamos el dispositivo de arranque, que en este caso es el USB que contiene la ISO. Una vez ya tenemos el ordenador encendido con el USB dentro configuramos paso por paso el nuevo SO(idioma, tipo de teclado, IPs...). Una vez con todo esto hecho ya podemos pasar al siguiente paso.

    Configuración del servidor en Proxmox

    Una vez Proxmox está instalado nos dará una IP, la cual será la que deberemos poner en el buscador de nuestro navegador para acceder a la pantalla de configuración, allí es dónde configuraremos nuestro servidor para la página web. Lo que nosotros hemos hecho a sido crear un contenedor, ya que es más fácil de manejar, i dentro poner una imagen del sistema operativo ubuntu. Ya dentro del contenedor instalamos el apache, el cual es el siguiente paso.

    Instalación de la píldora Apache

    Para instalar esta píldora dentro de nuestro contenedor pusimos el comando sudo apt install apache2, no hay mucho más que decir en esta parte asi que vamos a configurar esta píldora.

    Configuración de la píldora Apache

    Configurar esta píldora ha sido complicado para nosotros, principalmente porque daba muchos errores a la hora de poner el comando, además no guardaba los datos proporcionados al documento editado, pero al final conseguimos poder editar el apache con estos comandos: cd /var/www/gci/ y nano index.html.Una vez dentro del archivo, ponemos el código html. Una vez terminado el código, guardamos y cerramos el documento y nos disponemos a editar el archivo de configuración, concretamente el serveradmin y el nombre.

    Definiciones

    Proxmox

    Proxmox es un sistema operativo de código abierto que puede contener máquinas virtuales y contenedores para utilizarlas fácilmente, es por esto que es uno de los mejores SO para crear páginas web como estamos haciendo.

    BalenaEtcher

    Es una herramienta importante para instalar sistemas operativos en ordenadores, ya que este programa te permite flasehar la ISO(la imagen del sistema operativo) en un USB para más tarde usarla en otro ordenador y instalarle el SO prevamiente instalado en el USB.

    Apache

    Este es el servicio qué hemos usado para crear la página web, ya que nos permite, a través de una IP dada por la máquina, esta página sea accesible a todo el mundo
